# GIKI Connect

Campus-only social platform for GIK Institute — events, meetups, @username profiles, and in-app group chat.

## Repo layout

| Path | Description |
|------|-------------|
| `docs/GIKI-Connect-PRD.md` | Product requirements (source of truth) |
| `web/` | Next.js frontend + design preview |

## Frontend preview (port 3010)

```bash
cd web
npm install
npm run dev
```

- **Design hub:** http://localhost:3010/design
- **App shell:** http://localhost:3010

Port **3010** is used so **3000** stays free for other projects.
